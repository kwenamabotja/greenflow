# GreenFlow Gauteng: Inclusive Innovation for Clean Public Transport

## NCIC 2026 Submission - Gauteng Provincial Challenge

GreenFlow is a comprehensive Mobility-as-a-Service (MaaS) platform designed specifically for Gauteng, South Africa's economic powerhouse. As an entry in the National Cleantech Innovation Challenge (NCIC) 2026, this project demonstrates **Inclusive Innovation for Clean Public Transport** by combining real-time transit data, carbon tracking, and community-focused features to create equitable access to sustainable mobility.

## 🎯 Challenge Alignment

This project addresses Gauteng's transition needs in clean public transport by:

- **Inclusive Access**: Virtual taxi pooling and route optimization for underserved communities
- **Carbon Reduction**: Real-time CO₂ tracking and green wallet incentives
- **Community Empowerment**: Youth-focused features and local entrepreneurship support
- **Resilient Infrastructure**: Load-shedding aware routing and multi-modal integration
- **Innovation Pipeline**: Open-source platform for continuous improvement and scaling

## 🚀 Key Features

### Real-Time Transit Integration
- Gautrain, Metrobus, and taxi rank data
- Virtual taxi clustering for efficient ridesharing
- Live transit feeds with arrival predictions

### Carbon Impact Tracking
- Personal carbon footprint calculator
- Community leaderboard and incentives
- Green wallet system for carbon credits

### Inclusive Design
- Multi-language support (English, Afrikaans, Zulu)
- Accessible UI for all users
- Community-driven feature development

### Smart Routing
- Load-shedding aware route planning
- Multi-modal journey optimization
- Real-time traffic and transit updates

## 🏗️ Technical Architecture

```
Frontend (React + Vite)    API Server (Express)    Database (PostgreSQL)
├── Dashboard              ├── Transit APIs         ├── Transit Hubs
├── Route Planner          ├── Carbon Calculator    ├── Wallet Users
├── Green Wallet           ├── Power Integration    ├── Transactions
└── Transit Hubs           └── Authentication       └── Carbon Ledger
```

## 🛠️ Quick Start

1. **Setup Environment:**
   ```bash
   bash ./start.sh
   ```

2. **Start Development Servers:**
   ```bash
   # Terminal 1: API Server
   pnpm --filter @workspace/api-server run dev

   # Terminal 2: Frontend
   cd artifacts/greenflow && pnpm run dev
   ```

3. **Open Application:**
   ```
   http://localhost:5173
   ```
